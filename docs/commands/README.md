# Command Reference

This plugin ships the following public Fish commands:

- [`fissh`](fissh.md): SSH wrapper that tries to start `fish` on the remote host.
- [`mofish`](mofish.md): `mosh` wrapper for interactive remote shell sessions.
- [`kcd`](kcd.md): Namespace switcher for the current Kubernetes context.
- [`kctx`](kctx.md): Kubernetes context switcher with optional namespace assignment.
- [`git-bump`](git-bump.md): Semantic version tag bumping with optional dry runs and push control.
- [`repo`](repo.md): Repo navigation, clone bootstrap, and worktree entry point.
